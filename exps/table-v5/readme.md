### datasets-v1.py

生成的图片包含图像+部分语义信息。

将每个text进行类型判断，日期为蓝色、数字为绿色、页码为紫色、图片为黄色、文字为灰色。如此，获得的图片则是一张彩色图片，不同的文本框由于语义不同会展现不同的颜色，但是所有文字文本框都是灰色，所以只包含了部分的语义信息。

数据存入table-v3。



### datasets-v2.py

生成的图片包含图像信息。

将每个text设置为黑色，空白的部分设置为白色。如此，获得的图片只包含文本框的位置信息。

数据存入table-v4。



### datasets-v5.py 

生成的图片包含图像+全部语义信息。

将每个text进行分词，计算出所有的单词的词向量的平均值，然后用这个平均值作为这句话的句向量（3维）。如此，获得的图片则是一张彩色图片，不同的文本框由于语义不同会展现出不同的颜色。

数据存入table-v5。



### datasets-v6.py

生成的图片包含图像+全部语义信息。

将每个text进行分词，计算出每个词对应的位置，然后将该位置设置为该词向量（3维）对应的颜色。如从，获得的图片则是一张彩色图片，不同的单词由于与以不同会展现出不同的颜色。

数据存入table-v6。
